Что такое замыкание (closure) в JavaScript?

Нажмите для ответа

Ответ

Замыкание — это функция, которая запоминает своё лексическое окружение даже тогда, когда выполняется вне своей области видимости.

function outer() {
  let count = 0;
  return function() {
    return ++count;
  };
}
